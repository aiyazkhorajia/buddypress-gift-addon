Gift Buddypress Addons provide gift management functionality with buddypress plugin.

== Description ==

Gift Buddypress Addons provide gift management functionality with buddypress plugin with following features.

1. Admin can Add/Edit/Delete gifts and can manage gift category
2. Buddypress user can send or receive gift from profile page.


== Installation ==

how to install the plugin and get it working.

e.g.

1. Upload plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add new gifts  from WordPress admin and user will be able to see gift tab on profile.
